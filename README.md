# Crown Appliances — Field Diagnostics (prototype)

A clickable prototype of a guided appliance diagnostic tool for Chadwell Supply maintenance techs.

Enter or "scan" a Crown Appliances refrigerator model number, answer a couple of symptom questions, and the tool narrows down the likely cause and the exact part to order.

**Live demo:** replace with your GitHub Pages URL once deployed, e.g. `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Pilot models
- CR-1800
- CR-2400
- CR-3000

## Status
Prototype for internal pitch purposes. Decision-tree data (`TREE` object in `index.html`) uses placeholder causes/parts — swap in real failure data before using with actual techs.
